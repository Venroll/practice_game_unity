FindWayOut
2D-игра в жанре стелс-экшен, разработанная на Unity. Основная задача игрока — незаметно пробраться через уровень, избегая обнаружения противниками, собрать монеты и добраться до выхода.

Основные механики:
  Управление персонажем — свободное перемещение по уровню с видом сверху.
  Система обнаружения — противники имеют ограниченное поле зрения и замечают игрока только при отсутствии препятствий между ними.
  Искусственный интеллект — охранники патрулируют территорию, преследуют игрока при обнаружении и возвращаются к маршруту после потери цели.
  Взаимодействие с объектами — игрок может подбирать монеты, спрятанные на локации.
  Система победы и поражения — уровень считается пройденным после достижения выхода; при обнаружении игрока и последующей его поимке игра завершается поражением.
